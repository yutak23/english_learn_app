# 開発ロードマップ

英単語 SRS 学習アプリの開発優先順位とフェーズ計画です。

---

## 開発優先順位

### Phase 1: コア機能

**目標**: 基本的な学習機能を動作させる

1. 単語データ読み込み（複数ファイル、重複チェック）
2. SRS スケジューラ
3. 5単語1セット制の学習フロー
4. Study 画面（意味表示/非表示切り替え）
5. localStorage 保存・読み込み
6. セッション管理

### Phase 2: 学習体験向上

**目標**: 学習体験を豊かにする機能を追加

1. 音声再生
2. 時間計測
3. 出題優先順位（セット単位）
4. ローディング状態

### Phase 3: データ管理・レポート

**目標**: データ管理とレポート機能を実装

1. エクスポート/インポート
2. Report 画面
3. Settings 画面

### Phase 4: 拡張機能

**目標**: より高度な学習支援機能を追加

1. 苦手単語ピックアップ
2. ストリーク機能
3. 期間フィルタ
4. グラフ表示

---

## 将来の拡張可能性

以下の機能は現時点ではスコープ外ですが、将来的な実装候補です：

- **FSRS アルゴリズムへの移行**: より高度な SRS ロジック
- **カスタムカテゴリ/タグ機能**: 単語のグループ分け
- **学習目標設定**: 1日あたりの目標学習数
- **リマインダー通知**: Web Push API による学習リマインダー
- **オフライン対応強化**: Service Worker による完全オフライン動作
- **複数デバイス間の同期**: クラウドストレージ連携（オプション）
